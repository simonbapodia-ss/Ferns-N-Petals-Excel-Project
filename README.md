## Project Overview
This project analyzes sales data of Ferns N Petals to understand revenue trends, customer behavior, and product performance using Microsoft Excel.

## Objectives
- Analyze total orders and revenue
- Identify top-selling products and categories
- Understand seasonal and hourly sales trends
- Analyze city-wise order distribution

## Tools & Skills
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Filters
- Data Cleaning
- Dashboard Design

## Key Insights
- Highest revenue generated during festive occasions
- Certain product categories consistently outperform others
- Peak order hours identified for better planning
- Top cities contribute major share of orders
